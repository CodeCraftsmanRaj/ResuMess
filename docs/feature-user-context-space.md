# Feature: User Context Space

## Goal
Create the personalized user-specific source of truth for all future document generation.

## Scope
- Structured user profile storage
- Skills, projects, experience, achievements
- Reusable notes and facts
- PageIndex or vectorless RAG retrieval layer
- Context reuse across resume, CV, SOP, cover letter, and portfolio generation
